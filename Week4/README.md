# Week 4: Design Deep Dive, Fiori, Technical Design
## Reinforce Design + Technical Documentation (Morning) | Coursera (Afternoon)

This week covers **design deep dive** and **technical design** for an SAP Technology Consultant profile.

Morning sessions use the markdown files in this folder (learning + exercises). Afternoon sessions remain **self-paced Coursera** (see `coursera_study_plan.md`).

---

## Theme

- Design deep dive, including communication, pitching, and design review
- SAP Fiori deep dive and **fit-to-standard vs custom**
- Technical design tools and **developing a technical design solution**
- Short preview tying design back to implementation and lifecycle thinking
