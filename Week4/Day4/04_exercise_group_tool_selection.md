# Day 4: Group Exercise – Tool Selection by Project Context
## Two Groups, Two Project Contexts

Work in **2 groups**. Each group receives a project context and recommends which tools and ALM approach to prioritize.

---

## 👥 Group Setup

- Form **2 groups** (each group can have 2–3 members).
- In each group, assign roles:
  - **Facilitator** – guides the discussion and keeps time
  - **Recorder** – captures notes
  - **Presenter** – presents back to the class (~5 minutes)

Write your details:

- **Group number:** _____  
- **Your role:** ___________________________________________

---

## 🎯 Task Overview

For your project context:
1. Choose which tools you would prioritize (BAS, Integration Suite, Solution Manager, Cloud ALM, etc.)
2. Explain **why** (value, risk, maintainability, operations)
3. Mention what you would track in ALM (requirements, actions, tests, incidents)
