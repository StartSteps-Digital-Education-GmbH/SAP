# Day 1: Group Exercise – Process to SAP Platform Component
## Map a Business Process to One Platform Component and One HANA Capability

Work in a small group. Each group gets **one business process**. Your task is to map it to **one SAP platform component** and **one SAP HANA capability**, and present in about **5 minutes**.

---

## 👥 Group Setup

- Form **3 groups** (or 3 sub-groups if the class is large).
- Assign roles: **Facilitator**, **Recorder**, **Presenter**.
- Each group has a **different process** (see below).

**Your group number:** _____  
**Your role:** ___________________________________________

## 🎯 Task by Group

**For your assigned process:**
1. Name **one SAP platform component** (Fiori, ABAP, HANA Development, Cloud development) that would support this process and say **why** in one sentence.
2. Name **one SAP HANA capability** (e.g. real-time analytics, data modeling, predictive analytics, high-performance processing) that would help and say **why** in one sentence.
