# Day 3: Group Exercise – SAP Activate Phase Pairs
## Two Groups, Two Phase Pairs

Work in **2 groups**. Each group focuses on a **pair of SAP Activate phases** and prepares a short summary.

---

## 👥 Group Setup

- Form **2 groups** (each group can have 2–3 members).
- In each group, assign roles:
  - **Facilitator** – guides the discussion and keeps time
  - **Recorder** – captures key points
  - **Presenter** – presents back to the class (~5 minutes)

Write your details:

- **Group number:** _____  
- **Your role:** ___________________________________________

---

## 🎯 Task Overview

Each group takes **one pair** of phases:

- **Group 1:** Prepare + Explore  
- **Group 2:** Realize + Deploy (and think about how Run is prepared)  

For your phase pair:
1. List **3 key tasks** you would expect in those phases.
2. Name **one main deliverable** for each phase.
3. Describe **one risk** if that phase is done poorly or rushed.

Use `04_learning_sap_activate.md` as your reference.
