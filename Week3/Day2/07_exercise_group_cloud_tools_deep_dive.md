# Day 2: Group Exercise – Cloud Tools Deep Dive
## Three Tools, Three Groups

Work in **3 groups**. Each group takes one SAP cloud tool and prepares a short **“mini‑briefing”** for the rest of the class.

---

## 👥 Group Setup

- Form **3 groups**
- Assign roles in each group:
  - **Facilitator** – guides the discussion and keeps time
  - **Recorder** – captures key points
  - **Presenter** – presents your findings (~5 minutes)

Write your details:

- **Group number:** _____  
- **Your role:** ___________________________________________

## 🎯 Task Overview

Each group focuses on **one tool**:

- **Group 1:** SAP Business Technology Platform (BTP) / SAP Cloud Platform  
- **Group 2:** SAP Analytics Cloud (SAC)  
- **Group 3:** SAP Integration Suite  

For your tool:
1. Explain **what it is** (in simple terms).
2. Describe **when and why** a customer would use it.
3. Give **one concrete use case** that fits with today’s topics (cloud strategy, architecture, monitoring).

Use `03_learning_sap_cloud_architecture_tools.md` as your main reference.
