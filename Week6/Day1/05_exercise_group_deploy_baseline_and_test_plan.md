# Group Exercise: Baseline, Deploy, and Test Scripts Plan
## 2 Groups, 2–3 Members Each

Form **2 groups**. Assign roles:
- **Facilitator** – keeps group moving
- **Recorder** – writes deliverables clearly
- **Presenter** – presents to the class

Timebox:
- 15–20 minutes preparation
- 3–5 minutes per group presentation

---

## Scenario

A customer is moving forward with an SAP implementation. You have completed:
- Technical design approvals (Week 4)
- Planning for automation and environment management

Now the goal is to prepare something testable:
- stable baseline configuration
- deployed technical solution design
- test scripts ready for execution

---

## Group 1: Baseline + Integration Readiness

Deliverable: a “Baseline Readiness Checklist” with:
- 4 items that must be stable in baseline configuration
- 3 items that must be ready for integrations
- 1 monitoring/alerting consideration (to detect failures early)

## Group 2: Deploy Technical Design + Test Scripts Plan

Deliverable: a “Deploy + Test Scripts Plan” with:
- 3 deploy actions (what gets moved into a working setup)
- 3 test script preparation actions (what you must define before execution)
- 1 risk if deploy is delayed or incomplete

  2. ___________________________________________

- Test script prep actions (3):
  1. ___________________________________________

  ___________________________________________

---

## Present Back

Presenter should answer:
- What are your checklist items/actions?
- Which one item reduces the most risk?
- What do you need to confirm next with the team/customer?
